## 🖥️ Fish Decision Pro (FDP) v2.0.9

魚屋・鮮魚居酒屋向けの**経営管理ツール**です。

### 主な機能
- 🔪 **歩留まり原価計算** — 魚種別の歩留まり率・原価を自動計算
- 💹 **売上入力・管理** — 日別売上の記録と月次集計
- 📉 **ロス・廃棄管理** — 廃棄金額の記録と活用レシピ提案
- 📦 **在庫予測** — 仕入れ量の最適化を支援
- 📊 **分析ダッシュボード** — 売上・原価率・廃棄率をグラフ表示
- 🍳 **調理・売り方レシピ** — 業態別レシピカード
- 🔒 **PINロック** — 4桁PIN＋緊急マスター解除（🐟3タップ）
- 🔄 **自動アップデート確認** — GitHub連携でワンクリック更新
- ☁️ **DB配信** — 季節・業態別魚種DBをオンライン取得

### 動作環境
- ブラウザ：Chrome / Safari / Edge（最新版推奨）
- サーバー不要・完全オフライン動作
- データ保存：ブラウザのlocalStorage

---

## 🖥️ Fish POS Pro v8.0.0

魚屋・居酒屋向けの**タブレット型レジアプリ**です。

### 主な機能
- 💴 **会計・精算** — 現金・カード・QRコード等6種の決済対応
- 🪪 **会員番号管理** — 個人情報ゼロ・12桁匿名番号・スタンプカード
- ⭐ **スタンプ特典** — 来店回数に応じた特典を3段階設定
- 🏆 **常連ランキング** — 今月の来店数TOP20を匿名表示
- 🖨️ **レシート印刷** — window.print()対応（58mm/80mm）
- 📡 **Bluetooth印刷** — Android Chrome限定・サーマルプリンター直接送信
- 📢 **レシートCM** — レシート末尾にイベント告知を印刷
- 🔄 **自動アップデート確認** — GitHub連携でワンクリック更新
- 🎁 **テンプレート配信** — 季節・業態別スタンプ設定をオンライン取得
- 📅 **日次締め・月次レポート** — FDPへのデータ連携

### 動作環境
- ブラウザ：Chrome / Safari（最新版推奨）
- タブレット・スマートフォン・PC対応
- サーバー不要・完全オフライン動作
- データ保存：ブラウザのlocalStorage
